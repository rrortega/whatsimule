react-whatsapp-simulator

Simulador de conversaciones de WhatsApp en 3D interactivo con animaciones en tiempo real, indicador de tipeo y síntesis de efectos de sonido vía Web Audio API.

Interfaz Fiel a WhatsApp

Diseño en modo oscuro nativo, burbujas animadas con Framer Motion, aviso de cifrado y doble check azul.

🔊

Audio Sintetizado

Efectos de sonido procedimentales generados con Web Audio API al tipear, enviar y recibir mensajes sin assets .mp3.

🎨

Inclinación 3D en Scroll

Efecto de perspectiva tridimensional suave que reacciona de forma dinámica al hacer scroll en la página.

⚡

Multi-Framework

Soporte nativo para React / Next.js y adaptador CustomElement (<whatsapp-simulator>) para Vue, Svelte y HTML plano.

💻 Instalación y Uso Rápido

npm install react-whatsapp-simulator framer-motion lucide-react

import React from "react";
import { WhatsAppSimulator } from "react-whatsapp-simulator";
import "react-whatsapp-simulator/dist/whatsapp-simulator.css";

const scripts = {
  soporte: {
    id: "soporte",
    label: "Atención al Cliente AI",
    description: "Flujo de resolución de consultas 24/7",
    steps: [
      { sender: "user", type: "text", content: "Hola, necesito ayuda con mi pedido.", delay: 500 },
      { sender: "assistant", type: "text", content: "¡Hola! Con gusto te ayudo. ¿Me compartes tu número de orden?", delay: 1200 }
    ]
  }
};

export default function App() {
  return (
    <WhatsAppSimulator
      scripts={scripts}
      assistantName="Asistenxa Support"
      enable3DTilt={true}
      enableSound={true}
      locale="es"
    />
  );
}
